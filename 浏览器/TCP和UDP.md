### TCP和UDP

区别：

|          |                       TCP                        |              UDP               |
| :------: | :----------------------------------------------: | :----------------------------: |
|  可靠性  |                       可靠                       |             不可靠             |
|  连接性  |                     面向连接                     |             无连接             |
|   报文   |                    面向字节流                    |            面向报文            |
|   效率   |                    传输效率低                    |           传输效率高           |
|  双工性  |                    全双工通信                    | 一对一、一对多、多对一、多对多 |
| 流量控制 |                     滑动窗口                     |               无               |
| 拥塞控制 |         慢开始、拥塞避免、快重传、快恢复         |               无               |
| 传输速度 |                        慢                        |               快               |
| 应用场景 | 对效率要求低，对准确性要求高或者要求有连接的场景 |  对效率要求高，对准确性要求低  |
